# Évolutions Futures — Roadmap

## Court terme (3-6 mois)

### Application mobile (iOS/Android)
- Application React Native
- Synchronisation avec le client desktop
- Notifications push
- Scan QR pour activation licence

### SSO / OAuth2
- Connexion Google, GitHub, Microsoft
- Gestion centralisée des identités
- Compatible OpenID Connect

## Moyen terme (6-12 mois)

### API publique
- Documentation OpenAPI/Swagger
- Rate limiting par clé API
- Dashboard développeurs
- Webhooks sortants

### Système de paiement
- Intégration Stripe
- Abonnements mensuels/annuels
- Factures PDF automatiques
- Réduction volume (licences multiples)

## Long terme (12-24 mois)

### Intelligence Artificielle
- Chatbot support client
- Analyse prédictive pannes
- Recommandations personnalisées
- Détection automatique d'anomalies

### Haute disponibilité complète
- Multi-région
- Load balancing global (Anycast)
- Réplication multi-master PostgreSQL
- Délestage automatique

## Idées supplémentaires

- Marketplace de modules
- Intégration CI/CD pour modules personnalisés
- Plugin système avec SDK
- Portail revendeurs (white-label)
- API temps réel pour IoT
- Audit RGPD automatisé
